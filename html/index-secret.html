Echange de secret sécurisé

Mise en garde

Vous allez consulter un secret qui va s'autodétruire. Une fois ce secret consulté, vous ne pourrez plus réutiliser le lien à nouveau. Si vous souhaitez le conserver, mettez-le dans un lieu sécurisé ou dans un gestionnaire de mots de passe sécurisé.

Mise en garde

Vous allez consulter un secret qui peut être consulté pendant une durée pré-définie. Lorsque cette durée sera écoulée, vous ne pourrez plus réutiliser le lien à nouveau. Si vous souhaitez le conserver, mettez-le dans un lieu sécurisé ou dans un gestionnaire de mots de passe sécurisé.

Mise en garde

Vous allez consulter un secret qui peut être consulté jusqu'à ce qu'il soit supprimé. Lorsque le secret sera supprimé, vous ne pourrez plus réutiliser le lien à nouveau. Si vous souhaitez le conserver, mettez-le dans un lieu sécurisé ou dans un gestionnaire de mots de passe sécurisé.

Mise en garde

Vous allez consulter un secret qui peut être consulté indéfiniement. Si vous souhaitez le conserver aillers, mettez-le dans un lieu sécurisé ou dans un gestionnaire de mots de passe sécurisé.
